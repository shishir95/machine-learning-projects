# Analysis of Virat Kohli's Batting Performance and Pitch Reports

This project analyzes the batting performance of cricketer Virat Kohli using publicly available data and explores insights from pitch reports through text analysis. The goal is to provide a comprehensive overview of his career statistics and identify trends and patterns in his batting and the pitches he has played on.

**Data Sources:**

- **Batting Data:** Web scraped from a cricket statistics website.
- **Text Data:** Manually collected pitch reports from DOCX files.
